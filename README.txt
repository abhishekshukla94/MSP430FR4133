------C Code for MSP430FR4133 Launchpad------

Code Composer Studio v9.1.0

About:

From this piece of code, you will be able to blink the RED LED which is located at P1.0 with the help of Button located at P1.2.

Note: Make sure while, peforming the below steps, you should have MSP430FR4133 connected to PC.

1. Copy and Paste the Code in CCS.
2. Click on the Build button to build the project.
3. Click on Debugger option to finally upload the sketch to your MSP430FR4133.
